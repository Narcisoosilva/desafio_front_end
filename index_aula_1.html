Tags em Bloco

<header>: Define o cabeçalho da página, normalmente contendo títulos e elementos de navegação.

<h1>: Define um título de nível 1, usado para o título principal da página.

<nav>: Define uma seção de navegação.

<ul>: Cria uma lista não ordenada.

<li>: Define um item de lista.

<main>: Contém o conteúdo principal do documento.

<section>: Define uma seção de conteúdo tematicamente relacionado.

<h2>: Define um título de nível 2, usado para subtítulos.

<p>: Define um parágrafo de texto.

<footer>: Define o rodapé da página.

Explicação das tags adicionais

<!DOCTYPE html>: Define o documento como um HTML5.

<html lang="pt-BR">: Elemento raiz do documento HTML. O atributo lang define o idioma da página.

<head>: Contém meta-informações sobre o documento, como o título e links para scripts e folhas de estilo.

<meta charset="UTF-8">: Define a codificação de caracteres do documento.

<meta name="viewport" content="width=device-width, initial-scale=1.0">: Faz com que a página seja responsiva.

<title>: Define o título da página que aparece na aba do navegador.

<link rel="stylesheet" href="styles.css">: Vincula um arquivo CSS externo à página.

<body>: Contém o conteúdo visível da página.

<script src="scripts.js"></script>: Inclui um arquivo JavaScript externo.
